<a name="jquery-cycle-288"></a>jQuery 2,88 ciclo
====================
Os seguintes arquivos ciclo jQuery são hospedados neste CDN:

- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.All.js &mdash; a versão completa do plug-in do jQuery ciclo. Inclui todos os mais efeitos especiais de vinte e todas as opções.
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.All.min.js &mdash; a versão minimizada do arquivo acima. Use esta versão para aplicativos de produção para melhorar o desempenho do site.
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.js &mdash; nesta versão do plug-in do jQuery ciclo é menor e contém apenas o efeito de transição "desaparecer".
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.min.js &mdash; a versão minimizada do arquivo acima. Use esta versão para aplicativos de produção para melhorar o desempenho do site.
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.Lite.js &mdash; o jQuery ciclo plug-in que contém somente a transição de "desaparecer" sem todas as opções de uma versão ainda menor.
- http://AJAX.aspnetcdn.com/AJAX/jQuery.Cycle/2.88/jQuery.Cycle.Lite.min.js &mdash; a versão minimizada do arquivo acima. Use esta versão para aplicativos de produção para melhorar o desempenho do site.
