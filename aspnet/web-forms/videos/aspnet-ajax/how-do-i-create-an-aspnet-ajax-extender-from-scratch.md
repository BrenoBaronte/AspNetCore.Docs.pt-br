====================
por [Carlos Pels](https://twitter.com/chrispels)

Saiba como criar um extensor AJAX do ASP.NET para um controle de servidor padrão do ASP.NET. Nós são mostrados como adicionar propriedades do lado do servidor e o JavaScript do lado do cliente para estender o comportamento do controle de caixa de texto padrão, embora a mesma abordagem pode ser aplicada a qualquer outro controle de servidor.
