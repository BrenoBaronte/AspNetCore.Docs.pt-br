<a name="how-do-i-use-the-aspnet-ajax-profile-services"></a>[Como fazer:] Usar os serviços de perfil do ASP.NET AJAX?
====================
por [Carlos Pels](https://twitter.com/chrispels)

O ASP.NET 2.0 introduzida [propriedades de perfil](https://msdn.microsoft.com/library/at64shx3.aspx) com a qual você pode armazenar os detalhes para cada usuário do seu aplicativo Web do ASP.NET. Serviços de perfil do ASP.NET AJAX permitem que a sua interface de Web do lado do cliente recuperar propriedades de perfil do usuário do aplicativo Web do lado do servidor. Este vídeo demonstra como um site de exemplo habilitado para AJAX viagem pode armazenar as informações de solicitação de viagem de um usuário para que os detalhes podem ser recuperados quando ele ou ela próxima visita o site.
