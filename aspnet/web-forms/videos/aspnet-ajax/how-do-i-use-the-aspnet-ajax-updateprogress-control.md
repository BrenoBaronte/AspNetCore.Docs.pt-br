<a name="how-do-i-use-the-aspnet-ajax-updateprogress-control"></a>[Como fazer:] Use o controle UpdateProgress do ASP.NET AJAX?
====================
por [Joe Stagner](https://github.com/JoeStagner)

Em um aplicativo AJAX normalmente há uma pausa notável entre a hora em que o navegador faz uma solicitação de rede e a hora em que o navegador recebe uma resposta. Para informar o usuário que a conversa de rede está ocorrendo, o ASP.NET AJAX UpdateProgress controle fornece comentários imediatos, exibindo uma imagem animada, 'Carregar' texto ou outros HTML que desaparece quando a resposta de rede é recebida.
