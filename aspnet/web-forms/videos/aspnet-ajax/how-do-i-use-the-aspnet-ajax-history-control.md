<a name="how-do-i-use-the-aspnet-ajax-history-control"></a>[Como fazer:] Use o controle de histórico do ASP.NET AJAX?
====================
por [Scott Golightly](https://github.com/scottgolightly)

Um problema que enfrentam um aplicativo típico do AJAX é que uma atualização parcial de página não é adicionada ao histórico do navegador da Web. Isso significa que o botão Voltar do navegador não voltar uma etapa de AJAX, mas volta um documento inteiro, que é provavelmente não espera que o usuário. O controle de histórico do AJAX ASP.NET permite ao desenvolvedor inserir pontos de histórico para que o usuário pode clicar no navegador botões Voltar e Avançar para mover entre etapas de AJAX.
