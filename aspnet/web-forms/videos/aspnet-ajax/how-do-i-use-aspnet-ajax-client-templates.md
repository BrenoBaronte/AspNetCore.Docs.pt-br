<a name="how-do-i-use-aspnet-ajax-client-templates"></a>Como fazer: usar ASP.NET AJAX cliente modelos
====================
por [Rob Windsor](https://twitter.com/robwindsor)

Há duas etapas principais envolvidas na criação de interações de AJAX, chamada de serviços de forma assíncrona e atualizar a página com os dados resultantes. ASP.NET AJAX sempre fez a primeira parte fácil, mas ele não foi fornecido mais suporte para o segundo. Para resolver esse problema, a Microsoft adicionou modelos do lado do cliente e associação de dados e neste vídeo, Rob Windsor demonstrará como usar esses novos recursos em seus aplicativos AJAX.
