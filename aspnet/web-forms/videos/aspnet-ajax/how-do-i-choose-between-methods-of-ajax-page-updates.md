<a name="how-do-i-choose-between-methods-of-ajax-page-updates"></a>[Como fazer:] Escolha entre os métodos do AJAX página atualizações?
====================
por [Joe Stagner](https://github.com/JoeStagner)

Neste vídeo, Joe Stagner compara os dois métodos principais de execução de atualizações de página de estilo AJAX em um aplicativo ASP.NET. O primeiro método é usar um UpdatePanel, em que nenhum código adicional precisa ser gravada no lado do cliente ou servidor. A vantagem de usar o UpdatePanel é tudo funciona automaticamente. A penalidade é que, no cliente requer muitos dados a serem incluídas na solicitação AJAX e da resposta e no servidor exige um ciclo de vida da página inteira a ser executado. O segundo método é usar retornos de chamada de rede, em que o código adicional precisa ser gravada no lado do cliente e do lado do servidor. A vantagem de usar retornos de chamada de rede é que no cliente requer muito pouco dados a serem incluídos na solicitação AJAX e da resposta e o servidor requer somente o método de chamada de serviço a ser executado. O penality é o tempo e esforço necessário para gravar o código necessário. Joe conclui o vídeo discutindo o que você deve considerar ao escolher entre os dois métodos principais de atualizações da página de estilo AJAX. (Este vídeo usa o código do [como começar com o ASP.NET AJAX](how-do-i-get-started-with-aspnet-ajax.md) vídeo e o [como faço para tornar cliente rede retornos de chamada com o ASP.NET AJAX](how-do-i-make-client-side-network-callbacks-with-aspnet-ajax.md) vídeo.)
