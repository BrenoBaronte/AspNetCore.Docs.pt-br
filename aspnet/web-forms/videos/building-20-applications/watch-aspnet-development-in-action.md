<a name="watch-aspnet-development-in-action"></a>Desenvolvimento do ASP.NET de inspeção em ação
====================
por [Joe Stagner](https://github.com/JoeStagner)

Observe como é fácil criar um "para fazer" do zero usando ASP.NET 2.0, o ASP.NET AJAX, o SQL Server Express e o Visual Web Developer do aplicativo de lista de tarefas. Joe Stagner da Microsoft começa criando um novo banco de dados e o código de acesso aos dados. Em seguida, vemos como definir uma página de lista de tarefas que exibe e atualiza o conteúdo no banco de dados. Por fim, saiba como adicionar funcionalidade AJAX para a página para melhorar a experiência do usuário.
