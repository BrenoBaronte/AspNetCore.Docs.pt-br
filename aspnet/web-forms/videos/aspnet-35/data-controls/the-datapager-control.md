<a name="the-datapager-control"></a>O controle DataPager
====================
por [Joe Stagner](https://github.com/JoeStagner)

Geralmente, o controle DataPager é usado com o controle ListView. O controle ListView é controlada por dados. Primeiro, crie um controle ListView e fornecê-la alguns dados. Em seguida, descarte o controle DataPager até a página para aprimorar a página da web funcionalmente. Se você não tiver observados ListView controle como faço vídeo seria melhor para examiná-la antes de usar o controle DataPager.
