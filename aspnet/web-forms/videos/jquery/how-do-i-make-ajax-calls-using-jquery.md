<a name="how-do-i-make-ajax-calls-using-jquery"></a>Como posso Fazer chamadas Ajax usando jQuery?
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostra como fazer chamadas Ajax usando a biblioteca jQuery. É criado um site da web de exemplo que contém um serviço web simples que retorna um valor. Em seguida, o método de ajax() jQuery é usado para chamar o serviço web. Um número das opções do método ajax() é definido como url, tipo de dados, dados, etc. Além disso, os manipuladores de eventos para condições de êxito e erro são definidas. Serviços web do outro, em seguida, é definido que aceita um argumento e também retorna um valor. Uma segunda chamada ajax() é definida que transmite um valor para a chamada de serviço web e processa os dados de retorno. Por fim, a configuração de opções globais e manipuladores de eventos para chamadas de método ajax() são discutidos.
