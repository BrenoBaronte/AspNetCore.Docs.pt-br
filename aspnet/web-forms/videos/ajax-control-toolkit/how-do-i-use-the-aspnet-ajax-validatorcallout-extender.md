<a name="how-do-i-use-the-aspnet-ajax-validatorcallout-extender"></a>[Como fazer:] Usar o extensor ValidatorCallout do ASP.NET AJAX?
====================
por [Joe Stagner](https://github.com/JoeStagner)

Os controles de validação ASP.NET padrão funcionam no servidor e cliente. Validação do lado do servidor adiciona a segurança de seu aplicativo Web, enquanto adiciona de validação do lado do cliente para a facilidade de uso do seu aplicativo Web. Este vídeo demonstra como o controle ValidatorCallout do ASP.NET AJAX Control Toolkit estende os controles de validação padrão para aprimorar a experiência do usuário, posicionamento claramente as mensagens de validação nas caixas de texto explicativo.
