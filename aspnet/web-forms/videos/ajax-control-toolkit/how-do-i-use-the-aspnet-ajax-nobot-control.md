<a name="how-do-i-use-the-aspnet-ajax-nobot-control"></a>[Como fazer:] Use o controle NoBot do ASP.NET AJAX?
====================
por [Joe Stagner](https://github.com/JoeStagner)

O controle de NoBot do Kit de ferramentas de controle AJAX ASP.NET ajuda a impedir que certos tipos de acesso não autorizado aos seus aplicativos Web. Neste vídeo, podemos ver como o controle NoBot oferece inúmeras internos protege contra acesso automatizado, inclusive verificando o número e a frequência de envios de formulário. Além disso, podemos ver como o controle permite que o desenvolvedor definir um desafio personalizado, como um teste de JavaScript que só terá êxito se a página é exibida em um navegador.
