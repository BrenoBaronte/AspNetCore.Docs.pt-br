<a name="how-do-i-use-the-aspnet-ajax-maskededit-controls"></a>[Como fazer:] Use os controles MaskedEdit do ASP.NET AJAX?
====================
por [Joe Stagner](https://github.com/JoeStagner)

O ASP.NET AJAX Control Toolkit fornece o controle MaskedEdit e o controle MaskedEditValidator que funcionam em conjunto para solicitar, estilo e validar os dados que o usuário insere em uma caixa de texto. Isso permite que a página da Web fornecer ao usuário feedback visual imediato no fato dos dados inseridos de um tipo válido e o formato.
