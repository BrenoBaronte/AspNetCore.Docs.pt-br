por [Joe Stagner](https://github.com/JoeStagner)

Este vídeo demonstra os conceitos básicos do controle guias do ASP.NET AJAX Control Toolkit, com a qual você pode criar um conjunto de guias para organizar o conteúdo em uma página da Web.
