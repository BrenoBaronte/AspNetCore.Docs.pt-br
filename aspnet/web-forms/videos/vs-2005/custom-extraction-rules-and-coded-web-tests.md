<a name="custom-extraction-rules-and-coded-web-tests"></a>Regras de extração personalizadas e testes Web codificados
====================
por Eric Lee

Visual Studio Team System torna fácil de testar um aplicativo Web gravando a interação do usuário com o aplicativo e, em seguida, repetindo repetidamente a interação gravada. Mas os usuários não normalmente executam as mesmas ações repetidamente. Neste vídeo, vemos como estender e personalizar nossos testes automatizados da Web para refletir com maior precisão a maneira que um usuário interage com o aplicativo. Aprendemos como criar regras de extração personalizada para extrair dados de um controle da Web que as regras de extração padrão podem não oferecer suporte e como adicionar c# ou testes Web codificados em Visual Basic para adicionar funcionalidade como loop, lógica e acessar dados externos.
