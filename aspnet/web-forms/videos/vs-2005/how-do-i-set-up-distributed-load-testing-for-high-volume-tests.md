<a name="how-do-i-set-up-distributed-load-testing-for-high-volume-tests"></a>[Como fazer:] Configurar o teste de carga distribuída para testes de alto Volume?
====================
por [Carlos Menegay](https://twitter.com/CMenegay)

O Visual Studio Team System permite que você configure distribuída de teste de carga, criando um teste "equipamento" que consiste em um controlador em um computador remoto e vários agentes no mesmos ou mais computadores remotos. Este vídeo orienta você instalar o controlador e os agentes e, em seguida, definindo os casos de teste para ser executado usando o controlador remoto. Como veremos, o Visual Studio Team System torna muito simples de configurar o teste para testes automatizados de alto volume de carga distribuída.
