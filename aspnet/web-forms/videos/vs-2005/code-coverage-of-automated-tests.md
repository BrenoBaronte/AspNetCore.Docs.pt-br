<a name="code-coverage-of-automated-tests"></a>Cobertura de código dos testes automatizados
====================
por Eric Lee

Este vídeo demonstra como usar a cobertura de código como uma métrica para determinar a eficiência de nossos testes são de um aplicativo Web ASP.NET. Após um teste, Visual Studio Team System informa a porcentagem de código que foi executado e indica o código executado com um plano de fundo verde e o código não executado com um plano de fundo vermelho, tornando mais fácil ver o código que foi coberto pelo teste. Você também aprenderá como você pode usar o c# ou Visual Basic para estender o teste automatizado.
