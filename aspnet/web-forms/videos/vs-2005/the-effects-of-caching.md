<a name="the-effects-of-caching"></a>Os efeitos de cache
====================
por Eric Lee

Quase todos os aplicativos da Web atuais precisa acessar dados de um banco de dados, mas essa operação comum pode ter um impacto real sobre o desempenho geral do aplicativo. Este vídeo mostra como o Visual Studio Team System pode ajudá-lo a identificar se o seu aplicativo está fazendo chamadas redundantes para o banco de dados e, em seguida, demonstra como você pode usar a funcionalidade de cache do ASP.NET 2.0 DataSource para eliminar as chamadas redundantes.
