<a name="how-do-i-write-code-more-quickly-with-unit-tests"></a>[Como fazer:] Escrever código mais rapidamente com testes de unidade?
====================
por [Carlos Menegay](https://twitter.com/CMenegay)

Neste vídeo, saiba como testes de unidade podem aumentar sua produtividade, facilitando o processo de depuração. Podemos ver como o Visual Studio Team System pode gerar um teste de unidade para automatizar a execução de código que está sendo depurado, que lhe poupa o tempo que leva para acionar o aplicativo Web e executar as etapas necessárias para executar manualmente o mesmo código de erro. O resultado de teste de unidade é que você pode escrever um código mais robusto, mais rapidamente.
