<a name="how-do-i-cache-portions-of-an-aspnet-page"></a>[Como fazer:] Cache de partes de uma página ASP.NET
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostra como armazenar em cache as partes de um ou mais de uma página ASP.NET implementando OutputCache para controles de usuário na página. Primeiro, uma página ASP.NET é criada e armazenado em cache usando a diretiva OutputCache. Vários controles de usuário, em seguida, são criados e armazenados em cache para durações diferentes. A relação entre a página em cache a duração e a duração do cache de controle é analisada. Os diferentes critérios para armazenar em cache os controles são examinados incluindo um exemplo de VaryByControl onde os controles de usuário aninhados podem controlar a cache de seus controles pai.
