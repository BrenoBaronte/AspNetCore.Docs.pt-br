<a name="how-do-i-work-with-master-pages-declaratively-and-programmatically"></a>[Como fazer:] Trabalhar com páginas mestras e de forma declarativa programaticamente
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostrar vários aspectos da definindo e usando páginas mestras em um site ASP.NET. Primeiro, consulte como criar páginas mestras declarativamente usando o modelo de página mestra ASP.NET e saber quais informações são ideal para uma página mestra em comparação a uma página de conteúdo individual. Em seguida, veja como atribuir programaticamente páginas mestras em rapidamente em uma página de conteúdo individual. Também aprenderá como criar uma classe de página de base que atribui uma página mestra programaticamente para todas as páginas que herdam da classe base. Além disso, consulte como definir a página mestra no arquivo Web. config. Por fim, saber como acessar elementos em uma página mestre em uma página de conteúdo individual, incluindo algumas práticas recomendadas para estruturar o acesso aos elementos da página mestra.
