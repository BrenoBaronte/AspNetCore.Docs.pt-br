<a name="how-do-i-implement-site-navigation-in-aspnet"></a>[Como fazer:] Implementar a navegação de Site no ASP.NET?
====================
por [Carlos Pels](https://twitter.com/chrispels)

Neste vídeo, saiba como definir um mapa do site e usá-lo com os controles de servidor do Menu, TreeView e mapa de site. Podemos ver como usar um mapa de site em uma página mestra menus de todo o site e como podemos pode ter um ou vários mapas do site em uma página. Além disso, podemos são mostrados como integrar um mapa de site com funções do ASP.NET usando "restrições de segurança" para mostrar ou ocultar as opções de menu para usuários com diferentes níveis de acesso dinamicamente.
