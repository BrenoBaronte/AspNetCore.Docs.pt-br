<a name="how-do-i-use-the-ability-to-create-groups-with-the-listview-control-for-different-data"></a>[Como fazer:] Use a capacidade para criar grupos com o controle ListView para dados diferentes
====================
por [Carlos Pels](https://twitter.com/chrispels)

Neste vídeo, Chris Pels mostrará como agrupar itens ao exibir dados em um controle ListView. Primeiro, consulte os fundamentos do layout do item no controle ListView com o LayoutTemplate e ItemTemplate. Em seguida, Aprenda como GroupTemplate se relaciona com o LayoutTemplate e ItemTemplate dentro de ListView para estabelecer um grupo de dados. Consulte como usar elementos DIV flutuantes para criar um layout tabular flexível porque nenhum dos modelos usados com ListView pode conter qualquer HTML. Isso proporciona flexibilidade significativa na criação de grupos de dados usando o controle ListView. Por fim, veja algumas opções para melhorar a aparência, manipulando os estilos CSS usados em modelos de ListView e como usar o GroupSeparatorTemplate, bem como o EmptyItemTemplate.
