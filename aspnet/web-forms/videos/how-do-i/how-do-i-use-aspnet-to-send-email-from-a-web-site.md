<a name="how-do-i-use-aspnet-to-send-email-from-a-web-site"></a>[Como fazer:] Usar o ASP.NET para Enviar Email de um Site da Web
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostra como criar e enviar um email em um site ASP.NET. Primeiro, saiba mais sobre as classes principais no namespace do mail para enviar um email em um site da web ASP.NET. Veja como criar uma mensagem de email usando a classe MailMessage inclusive como usar a classe MailAddress para o endereço de e como incluir um ou mais endereços. Em seguida, veja como o email é enviado usando a classe SmtpClient. Por fim, saiba mais sobre as opções adicionais para configurar a mensagem de email como CC, Cco, prioridade e adicionar anexos.
