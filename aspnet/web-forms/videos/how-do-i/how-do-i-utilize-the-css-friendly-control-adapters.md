<a name="how-do-i-utilize-the-css-friendly-control-adapters"></a>[Como fazer:] Utilize os adaptadores de controle amigável CSS?
====================
por [Carlos Pels](https://twitter.com/chrispels)

Os adaptadores de controle amigável CSS permitem que você alterar o HTML que é processado por um controle de servidor padrão do ASP.NET 2.0. Neste vídeo a saber é que o controle de Menu do ASP.NET padrão renderiza uma marca HTML `<table>` elemento, mas com os adaptadores de controle CSS amigável o mesmo controle processe um HTML `<ul>` elemento de lista, melhorando a acessibilidade e a apresentação do menu.
