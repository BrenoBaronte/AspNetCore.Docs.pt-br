<a name="how-do-i-configure-email-notification-for-health-monitoring-on-an-aspnet-web-site"></a>[Como fazer:] Configurar notificações por Email para um Site da Web ASP.NET de monitoramento de integridade
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostra como configurar notificações por email para um site da web ASP.NET de monitoramento de integridade. Primeiro, consulte como configurar o envio de email em um site da web ASP.NET com o uso do &lt;emailSettings&gt; elemento no arquivo Web. config. Em seguida, Aprenda como adicionar o SimpleMailWebEventProvider que envia emails para eventos como um provedor de monitoramento de integridade. Em seguida, consulte a eventos que podem ser usados com notificação por email ao examinar a configuração de monitoramento de integridade de nível de máquina de monitoramento de integridade padrão. Depois de revisar os eventos disponíveis, consulte implementada uma regra que mapeia a "todos os eventos" para o provedor de email. Ao iniciar o site da web vários emails, em seguida, são enviados e examinados quando elas são recebidas no Outlook. Por fim, alguns princípios básicos dos quais os eventos podem ser mapeados para o provedor de email de monitoramento de integridade são discutidos.
