<a name="how-do-i-determine-whether-to-use-a-web-site-or-a-web-application-project"></a>[Como fazer:] Determine se deseja usar um Site da Web ou um projeto de aplicativo Web
====================
por [Carlos Pels](https://twitter.com/chrispels)

Neste Chris Pels vídeo irá comparar as diferenças entre um projeto de site da web do ASP.NET e um projeto de aplicativo web. Aprenda as considerações para usar um tipo de projeto ou o outro. Primeiro, saiba mais sobre os atributos principais de um projeto de aplicativo web incluindo sua base de pasta. Em seguida, veja os atributos principais de um projeto de site da web e sua natureza de projeto com base tradicional do Visual Studio que fornece mais granularidade e flexibilidade, incluindo a adição de classes, lidar com namespaces e controles de usuário ou páginas de referência por meio de programação. O modelo de compilação para cada tipo de site da web, em seguida, é abordado junto com alternativas que incluem o uso do aspnet\_compiler.exe e aspnet\_merge.exe utilitários e o projeto de implantação de web do Visual Studio 2008.
