<a name="how-do-i-save-and-load-view-state-information-for-a-custom-web-server-control"></a>[Como fazer:] Salvar e informações de estado de exibição de carga para um controle de servidor Web personalizado
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostrará como manter e restaurar as informações de um controle de usuário personalizada no ViewState. Primeiro, saiba como manter valores simples que contém os tipos primitivos do .NET como etc booliano, cadeia de caracteres, Int32. Em seguida, consulte como persistir tipos complexos, como a classe que contêm propriedades de armazenamento de informações. A sequência de processamento das informações de estado de exibição durante o ciclo de vida de uma página ASP.NET também é abordada.
