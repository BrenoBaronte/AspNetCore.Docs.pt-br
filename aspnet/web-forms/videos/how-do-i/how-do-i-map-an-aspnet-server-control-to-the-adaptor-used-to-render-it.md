<a name="how-do-i-map-an-aspnet-server-control-to-the-adaptor-used-to-render-it"></a>[Como fazer:] Mapear um controle de servidor ASP.NET para o adaptador usado para renderizá-lo
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostrará como usar um adaptador de controle para fornecer diferentes renderizações para um controle de servidor ASP.NET sem realmente alterar controle em si. Neste vídeo, um controle BulletList do ASP.NET será adaptado para exibir cada item de lista horizontalmente usando elementos DIV em vez dos elementos UL tradicionais. Primeiro, consulte como criar uma classe que herda WebControlAdaptor e, em seguida, implementa o código para processar o novo formato de lista. Em seguida, Aprenda como mapear o novo adaptador de controle para o controle de servidor ASP.NET no arquivo de definição. browser. Em seguida, consulte como usar o novo adaptador de controle nas páginas em um site da web. Por fim, saiba como um adaptador de controle pode ser associado a todos os navegadores ou tipos específicos de navegadores.
