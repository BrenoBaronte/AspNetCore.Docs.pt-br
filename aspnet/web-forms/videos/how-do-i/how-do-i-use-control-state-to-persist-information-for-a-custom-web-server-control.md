<a name="how-do-i-use-control-state-to-persist-information-for-a-custom-web-server-control"></a>[Como fazer:] Estado de controle de uso para manter as informações para um controle de servidor Web personalizado
====================
por [Carlos Pels](https://twitter.com/chrispels)

Neste vídeo, Chris Pels mostrará como armazenar informações relacionadas a um controle de servidor web personalizado no estado do controle que é independente das informações de ViewState do controle. Ao contrário de ViewState o estado de controle não pode ser desativado e, portanto, é uma maneira mais confiável para armazenar informações importantes para um controle. Uma discussão sobre os tipos de informações são apropriados para armazenar o estado de controle também está incluída.
