<a name="how-do-i-use-skins-with-css-for-a-flexible-and-maintainable-aspnet-web-site"></a>[Como fazer:] Usar capas CSS para um Site da Web do ASP.NET flexível e sustentável
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostra como criar capas para controles de servidor ASP.NET e incorporar os estilos CSS para controlar a aparência dos controles mais sofisticadas. Primeiro, saiba como criar uma aparência para um controle de servidor ASP.NET, como um rótulo que controla a aparência dos rótulos ao longo de um site da web. Em seguida, consulte como abstrato a definição da aparência de uma capa em estilos CSS. Durante esse processo saiba as vantagens de ter as informações de estilo de uma capa nos estilos CSS. Em seguida, consulte como criar vários capas e estilos correspondentes para um único controle ASP.NET como um rótulo que pode ser usado para padronizar a aparência de certos tipos de rótulos em todo o site. Por fim, saiba as vantagens da aplicação de um estilo CSS para várias capas e a capacidade de aplicá-la a um elemento HTML genérico.
