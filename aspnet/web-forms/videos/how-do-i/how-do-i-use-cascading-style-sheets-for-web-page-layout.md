<a name="how-do-i-use-cascading-style-sheets-for-web-page-layout"></a>[Como fazer:] Usar folhas de estilo em cascata para Layout de página da Web?
====================
por [Carlos Pels](https://twitter.com/chrispels)

Neste vídeo, saiba como usar `<div>` elementos em combinação com folhas de estilo em cascata (CSS) para criar páginas da web de desempenho robusto e superior. O `<div>`abordagem baseada em será comparada com o `<table>` com base a abordagem de layout usada pela maioria dos sites da web ao longo de anos recentes. Em seguida, várias abordagens com base em CSS serão mostradas, variando de posicionamento absoluto para flutuante elementos. Também aprendemos os benefícios de separar HTML estrutural de design com o uso de CSS.
