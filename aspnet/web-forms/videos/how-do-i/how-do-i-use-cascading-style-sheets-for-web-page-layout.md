Neste vídeo, saiba como usar &amp;lt; div&amp;gt; elementos em combinação com folhas de estilo em cascata (CSS) para criar páginas da web de desempenho robusto e superior. O &amp;lt; div&amp;gt; abordagem baseada em será comparada com o &amp;lt; tabela&amp;gt; com base a abordagem de layout usada pela maioria dos sites da web ao longo de anos recentes. Também aprendemos os benefícios de separar HTML estrutural de design com o uso de CSS.
