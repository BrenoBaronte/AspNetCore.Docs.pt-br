<a name="how-do-i-create-and-call-a-simple-web-service-in-aspnet"></a>[Como fazer:] Criar e chamar um serviço Web simples no ASP.NET
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostrará como criar um serviço da web em ASP.NET e, em seguida, chamar o serviço de outro site da web ASP.NET. Primeiro, saiba como criar um componente. asmx que representa um serviço da web em um site da web ASP.NET. Quando criar o serviço web Aprenda a controlar vários aspectos da funcionalidade de serviços web com atributos do serviço web. Em seguida, veja como testar o serviço web na página de teste da web serviço Visual Studio. Em seguida, saiba como o ASP.NET permite que muitos tipos de dados do .NET a ser retornado de um serviço web. Criar um projeto de biblioteca de classe comercial e, em seguida, usar uma classe no projeto como um valor de retorno de um serviço web ao serializar a instância do objeto de negócios personalizada. Por fim, consulte como consumir um serviço web de um site da web ASP.NET.
