<a name="how-do-i-create-user-selectable-themes-for-a-web-site"></a>[Como fazer:] Criar temas selecionáveis pelo usuário para um Site da Web
====================
por [Carlos Pels](https://twitter.com/chrispels)

Neste vídeo, Chris Pels mostrará como criar vários temas para um site da web e, em seguida, permitir que o usuário selecionar o tema é aplicado ao site. Veja como criar uma interface do usuário simples que permite que o usuário selecione cada um dos temas com um controle de usuário. Além disso, saiba como definir um tema para uma página de conteúdo de um controle de usuário em uma página mestra.
