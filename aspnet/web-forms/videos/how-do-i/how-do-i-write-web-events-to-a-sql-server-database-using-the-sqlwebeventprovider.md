<a name="how-do-i-write-web-events-to-a-sql-server-database-using-the-sqlwebeventprovider"></a>[Como fazer:] Gravar eventos da Web em um banco de dados do SQL Server usando o SqlWebEventProvider
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostra como usar a SqlWebEventProvider de monitoramento de integridade do ASP.NET para registrar erros em um site para um banco de dados do SQL Server. Primeiro, Aprenda a função do provedor e dos eventos de monitoramento de integridade do ASP.NET. Em seguida, consulte como configurar um banco de dados do SQL Server com os objetos necessários para gravação de eventos usando o aspnet de monitoramento de integridade\_regiis utilitário, o utilitário mesmo usado para configurar a associação do ASP.NET. Em seguida, Aprenda como configurar o arquivo Web. config para registrar eventos em um site ASP.NET para o banco de dados do SQL Server recém-criado de monitoramento de integridade. Como parte dessa configuração, consulte como o arquivo Web. config de raiz no .NET Framework 2.0 tem provedores e eventos definidos que podem ser utilizados ao configurar o monitoramento de integridade. Esses conceitos básicos são uma base na qual os eventos personalizados podem ser criados de suas próprias informações específicas de log para um banco de dados do SQL Server em um site da web ASP.NET.
