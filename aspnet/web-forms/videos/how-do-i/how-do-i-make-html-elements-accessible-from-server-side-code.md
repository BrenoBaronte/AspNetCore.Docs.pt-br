<a name="how-do-i-make-html-elements-accessible-from-server-side-code"></a>[Como fazer:] Disponibilizar os elementos HTML do lado do código do servidor
====================
por [Carlos Pels](https://twitter.com/chrispels)

Neste vídeo Chris Pels mostrará como fazer elementos HTML padrão em uma página. aspx acessível a partir de código do lado do servidor para que possam ser usados no processamento como um controle de servidor ASP.NET padrão da página. Primeiro, consulte como a adição de um runat = "server" atributo a um controle HTML padrão como uma &lt;div&gt; permite que ele seja referenciado no código do lado do servidor com o Intellisense completo. Em seguida, veja alguns exemplos de como você pode manipular elementos HTML em uma página com essa técnica. Por fim, consulte como essa técnica pode ser usada em conjunto com o novo controle ListView no .NET Framework versão 3.5 para controlar o layout de uma lista.
