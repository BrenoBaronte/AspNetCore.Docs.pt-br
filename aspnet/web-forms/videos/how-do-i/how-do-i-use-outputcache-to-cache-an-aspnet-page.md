<a name="how-do-i-use-outputcache-to-cache-an-aspnet-page"></a>[Como fazer:] Use OutputCache para armazenar em Cache uma página ASP.NET
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostra como utilizar o armazenamento em cache interno do ASP.NET para armazenar uma página no cache, portanto, não precisa ser re-renderizados. Primeiro, a forma mais simples de cache de página é demonstrada em que a página inteira é armazenado em cache para um período de tempo especificado. Em seguida, uma abordagem mais granular é obtida em que os valores das variáveis de sequência de consulta são usados como base para o cache de páginas. Por fim, veja como fazer com que o tipo de navegador ser a base para o cache de uma página.
