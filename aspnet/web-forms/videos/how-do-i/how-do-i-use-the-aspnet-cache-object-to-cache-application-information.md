<a name="how-do-i-use-the-aspnet-cache-object-to-cache-application-information"></a>[Como fazer]: usar o objeto de Cache do ASP.NET para armazenar em Cache as informações do aplicativo
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostra como usar o objeto de Cache do ASP.NET para armazenar informações do aplicativo. Um exemplo de página é criado com uma interface do usuário simples e o código é adicionado, que preenche a grade com dados de um arquivo de disco XML. Em seguida, o código é revisado para armazenar em cache os dados no objeto de Cache de aplicativo depois que inicialmente é recuperado e lidas do cache para solicitações subsequentes. Por fim, os métodos são adicionados para tornar o uso de absolute e deslizando a dependência de tempo com base para o objeto armazenado em cache.
