<a name="how-do-i-use-a-hidden-field-to-store-and-manipulate-client-side-information"></a>[Como fazer:] Usar um campo oculto para armazenar e manipular informações de cliente
====================
por [Carlos Pels](https://twitter.com/chrispels)

Neste vídeo Chris Pels mostrará como usar um campo oculto como uma abordagem para o gerenciamento de estado simples em uma página ASP.NET e para comunicação de cliente informações sobre o processamento. Primeiro, consulte como adicionar um campo oculto para uma página usando um controle de servidor ASP.NET HiddenField e recuperar o valor do campo oculto no código do lado do servidor depois de um POST-back. Em seguida, Aprenda como manipular o valor do campo oculto no script do lado do cliente e, em seguida, recuperar o valor modificado no código do lado do servidor. Por fim, veja um exemplo de como usar vários campos ocultos para armazenar informações de processamento do lado do cliente que pode ser lançado para o processamento do servidor.
