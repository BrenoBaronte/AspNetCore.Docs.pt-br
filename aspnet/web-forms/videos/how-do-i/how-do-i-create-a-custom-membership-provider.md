<a name="how-do-i-create-a-custom-membership-provider"></a>[Como fazer:] Criar um provedor de associação personalizado?
====================
por [Carlos Pels](https://twitter.com/chrispels)

Um provedor de associação permite que um aplicativo web para armazenar e recuperar dados de associação para um usuário e o provedor de associação padrão do ASP.NET usa tabelas predefinidas do SQL Server. Neste vídeo, saiba como criar um provedor de associação personalizado que usa tabelas personalizadas do SQL Server separadas das tabelas predefinidas usadas pelo provedor padrão. Os princípios abordados o vídeo serão aplicada à criação de provedores de associação personalizado usando outros bancos de dados, como Access ou Oracle. Depois de aprender a criar o provedor de associação personalizado, é saber como configurar o provedor em um aplicativo web e ver uma demonstração de criação de novos usuários e autenticação do novo usuário com o provedor de associação personalizado.
