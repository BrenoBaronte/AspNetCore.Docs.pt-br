<a name="how-do-i-implement-view--models-to-manage-data-for-aspnet-mvc-views"></a>Como fazer: modelos de exibição implementar para gerenciar os dados para exibições do ASP.NET MVC?
====================
por [Carlos Pels](https://twitter.com/chrispels)

Neste Chris Pels vídeo mostra como usar o conceito de um "modelo de exibição" para gerenciar os dados para exibições do ASP.NET MVC. Primeiro, é criado um projeto MVC de exemplo que usa um ViewDataDictionary para empacotar e transmitir dados de um controlador para um modo de exibição. Em seguida, uma classe é criada que representa um modelo de exibição para um modo de exibição do projeto de exemplo MVC. Como parte dessa implementação, as vantagens da abordagem do modelo de exibição são discutidas. Por fim, há uma discussão sobre como particionar a lógica do aplicativo ao usar uma implementação do modelo de exibição.
