<a name="how-do-i-work-with-model-binders-in-an-mvc-application"></a>Como fazer: usar associadores de modelo em um aplicativo MVC?
====================
por [Carlos Pels](https://twitter.com/chrispels)

Este Chris Pels vídeo mostra como usar associadores de modelo para associar os dados de formulário para objetos comerciais complexas em modelos em um aplicativo MVC. Primeiro, um aplicativo MVC é criado junto com um modelo simples para uma classe de cliente junto com um controlador e a exibição de edição de dados de cliente. Em seguida, a associação automática de modelos do MVC é demonstrada com a classe de cliente. Em seguida, o atributo de associação é discutido e demonstrou como um mecanismo para controlar os detalhes do processo de associação para incluir ou excluir propriedades específicas de um objeto complexo. Por fim, o uso de prefixos de namespace para a segmentação de dados do formulário em uma página é abordado.
