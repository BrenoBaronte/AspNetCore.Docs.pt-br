<a name="bundling-and-minifying-assets-in-an-aspnet-web-pages-razor-site"></a>Empacotando e minimizando os ativos em um Site de páginas da Web (Razor) do ASP.NET
====================
por [Microsoft](https://github.com/microsoft)

> Empacotamento e minimização são maneiras de acelerar seu site. Agrupamento permite que você combine várias JavaScript (*. js*) arquivos ou em várias folhas de estilo em cascata (*. CSS*) arquivos para que eles podem ser baixados como uma unidade, em vez de um de cada vez. Minimização comprime limite de espaço em branco e executa outros tipos de compactação para tornar os arquivos baixados como pequeno uma possível.
> 
> > [!NOTE]
> > A versão RC 2 de páginas da Web do ASP.NET não dá suporte a empacotamento e minimização porque o pacote que contém os elementos necessários ainda não está disponível no Microsoft WebMatrix. Pedimos desculpas pelo inconveniente. O pacote deve estar disponível na versão final de 2 de páginas da Web do ASP.NET e o WebMatrix 2.
