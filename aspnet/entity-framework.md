<a name="entity-framework"></a>Entity Framework
====================
Entity Framework (EF) é um mapeador relacional de objeto que permite aos desenvolvedores de .NET trabalhar com dados relacionais usando objetos específicos do domínio. Elimina a necessidade para a maioria do código de acesso a dados que os desenvolvedores geralmente precisam gravar.


- [Obtê-lo:](https://msdn.com/data/ee712906) adicionar isso ao seu projeto e começar a aproveitar esse poderoso O/RM.
- [Introdução:](https://msdn.com/data/ee712907) visite o guia para vídeos e tutoriais que ajudarão você a aproveitar seus dados.
- [Get-Help:](https://msdn.com/data/hh913619) saber como obter ajuda de especialistas e contribuir para a comunidade de dados sobre estouro de pilha.
- [Contribuir:](https://github.com/aspnet/EntityFramework6) Descubra como você pode ajudar a aprimorar o Entity Framework, um modelo de desenvolvimento aberto, mesmo visitando o site do GitHub.
