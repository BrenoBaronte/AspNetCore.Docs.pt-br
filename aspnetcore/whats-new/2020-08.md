# <a name="aspnet-core-docs-whats-new-for-august-1-2020---august-31-2020"></a>ASP.NET Core docs: o que há de novo em 1º de agosto de 2020 a 31 de agosto de 2020

Bem-vindo ao que há de novo no ASP.NET Core docs de 1º de agosto de 2020 até 31 de agosto de 2020. Este artigo lista algumas das principais alterações feitas no docs durante esse período.

## <a name="blazor"></a>Blazor

### <a name="new-articles"></a>Novos artigos

- <xref:blazor/test> -Tópico de teste mais incrivelmente
- <xref:blazor/webassembly-lazy-load-assemblies> -Tema de assemblies de carregamento lento do WASM

### <a name="updated-articles"></a>Artigos atualizados

- <xref:blazor/call-web-api> -Mova a seção HttpClient e HttpRequestMessage para obter maior visibilidade
- <xref:blazor/components/data-binding> -Aprimoramentos de parâmetros e Associação de componentes
- <xref:blazor/debug> -Mais boas atualizações do WASM de depuração: diretrizes de Visual Studio para Mac
- <xref:blazor/forms-validation>
  - Componentes de InputRadio/grupo de mais incrivelmente
  - Atualizações de validação de formulário mais incrivelmente
- <xref:blazor/host-and-deploy/webassembly> -Adicionar diretrizes para uma implantação hospedada com vários aplicativos de WASM de mais incrivelmente
- <xref:blazor/security/webassembly/additional-scenarios> -Alta cobertura de solicitação de saída de segurança WASMda
- <xref:blazor/security/webassembly/aad-groups-roles> -O comportamento do documento na segurança de WASM mais alta quando o AAD encontra mais de 5 funções ou grupos
- <xref:blazor/state-management> -Adicionar orientação sobre o estado persistente entre sessões do navegador
