# <a name="migrate-from-aspnet-core-31-to-50"></a>Migrar de ASP.NET Núcleo 3.1 para 5.0

Por [Scott Addie](https://github.com/scottaddie)

Este artigo explica como atualizar um projeto existente do núcleo 3.1 ASP.NET para ASP.NET O Core 5.0.

> [!IMPORTANT]
> ASP.NET Core 5.0 está atualmente em pré-visualização.

## <a name="update-net-core-sdk-version-in-globaljson"></a>Atualizar a versão do SDK do .NET Core em global.json

Se você confiar em um arquivo [global.json](/dotnet/core/tools/global-json) para segmentar uma `version` versão específica do .NET Core SDK, atualize a propriedade para a versão .NET 5.0 SDK instalada. Por exemplo:

```diff
{
  "sdk": {
-    "version": "3.1.200"
+    "version": "5.0.100-preview.2.20176.6"
  }
}
```

## <a name="update-the-target-framework"></a>Atualize a estrutura de destino

No arquivo do projeto, atualize o [Referencial de Destino (TFM)](/dotnet/standard/frameworks) para `netcoreapp5.0`:

```diff
<Project Sdk="Microsoft.NET.Sdk.Web">

  <PropertyGroup>
-    <TargetFramework>netcoreapp3.1</TargetFramework>
+    <TargetFramework>netcoreapp5.0</TargetFramework>
  </PropertyGroup>

</Project>
```

## <a name="update-package-references"></a>Referências do pacote de atualização

No arquivo do projeto, `Microsoft.Extensions.*` atualize `Version` cada `Microsoft.AspNetCore.*` atributo de referência do pacote para 5.0.0 ou posterior. Por exemplo:

```diff
<ItemGroup>
-    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="3.1.2" />
-    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="3.1.2" />
+    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="5.0.0-preview.2.20167.3" />
+    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="5.0.0-preview.2.20160.3" />
</ItemGroup>
```

## <a name="update-docker-images"></a>Atualizar imagens do Docker

Para aplicativos que usam o Docker, use uma imagem base que inclua ASP.NET Core 5.0. Por exemplo:

```bash
docker pull mcr.microsoft.com/dotnet/core/aspnet:5.0
```

## <a name="review-breaking-changes"></a>Revisar mudanças de quebra

Para obter alterações de .NET Core 3.1 para .NET 5.0, consulte [Alterações de violação da migração da versão 3.1 para 5.0](/dotnet/core/compatibility/3.1-5.0). ASP.NET Núcleo e Núcleo de Entidades também estão incluídos na lista.
