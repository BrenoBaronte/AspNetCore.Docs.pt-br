# <a name="host-aspnet-core-in-a-web-farm"></a><span data-ttu-id="ebcf4-107">Hospedar o ASP.NET Core em um web farm</span><span class="sxs-lookup"><span data-stu-id="ebcf4-107">Host ASP.NET Core in a web farm</span></span>

<span data-ttu-id="ebcf4-108">Por [Chris Ross](https://github.com/Tratcher)</span><span class="sxs-lookup"><span data-stu-id="ebcf4-108">By [Chris Ross](https://github.com/Tratcher)</span></span>

<span data-ttu-id="ebcf4-109">Um *web farm* é um grupo de dois ou mais servidores web (ou *nós*) que hospedam várias instâncias de um aplicativo.</span><span class="sxs-lookup"><span data-stu-id="ebcf4-109">A *web farm* is a group of two or more web servers (or *nodes*) that host multiple instances of an app.</span></span> <span data-ttu-id="ebcf4-110">Quando as solicitações de usuários chegam a um web farm, um *balanceador de carga* distribui as solicitações para os nós de farm da web.</span><span class="sxs-lookup"><span data-stu-id="ebcf4-110">When requests from users arrive to a web farm, a *load balancer* distributes the requests to the web farm's nodes.</span></span> <span data-ttu-id="ebcf4-111">Os web farms melhoram:</span><span class="sxs-lookup"><span data-stu-id="ebcf4-111">Web farms improve:</span></span>

* <span data-ttu-id="ebcf4-112">**Confiabilidade/disponibilidade**: quando um ou mais nós falham, o balanceador de carga pode rotear solicitações para outros nós de funcionamento para continuar processando solicitações.</span><span class="sxs-lookup"><span data-stu-id="ebcf4-112">**Reliability/availability**: When one or more nodes fail, the load balancer can route requests to other functioning nodes to continue processing requests.</span></span>
* <span data-ttu-id="ebcf4-113">**Capacidade/desempenho**: vários nós podem processar mais solicitações do que um único servidor.</span><span class="sxs-lookup"><span data-stu-id="ebcf4-113">**Capacity/performance**: Multiple nodes can process more requests than a single server.</span></span> <span data-ttu-id="ebcf4-114">O balanceador de carga equilibra a carga de trabalho ao distribuir as solicitações para os nós.</span><span class="sxs-lookup"><span data-stu-id="ebcf4-114">The load balancer balances the workload by distributing requests to the nodes.</span></span>

* <span data-ttu-id="ebcf4-117">**Manutenção**: os nós de um Web farm podem contar com um conjunto de serviços compartilhados, o que resulta em um gerenciamento de sistema mais fácil.</span><span class="sxs-lookup"><span data-stu-id="ebcf4-117">**Maintainability**: Nodes of a web farm can rely on a set of shared services, which results in easier system management.</span></span> <span data-ttu-id="ebcf4-118">Por exemplo, os nós de um web farm podem contar com um servidor de banco de dados individual e um local de rede comum para recursos estáticos, como imagens e arquivos para download.</span><span class="sxs-lookup"><span data-stu-id="ebcf4-118">For example, the nodes of a web farm can rely upon a single database server and a common network location for static resources, such as images and downloadable files.</span></span>

<span data-ttu-id="ebcf4-119">Este tópico descreve as configurações e dependências para aplicativos ASP.NET Core hospedados em um web farm que conta com recursos compartilhados.</span><span class="sxs-lookup"><span data-stu-id="ebcf4-119">This topic describes configuration and dependencies for ASP.NET core apps hosted in a web farm that rely upon shared resources.</span></span>
