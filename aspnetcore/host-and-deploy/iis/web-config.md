## <a name="configuration-of-aspnet-core-module-with-webconfig"></a><span data-ttu-id="933d3-124">Configuração do módulo ASP.NET Core com `web.config`</span><span class="sxs-lookup"><span data-stu-id="933d3-124">Configuration of ASP.NET Core Module with `web.config`</span></span>

<span data-ttu-id="933d3-126">O arquivo a seguir `web.config` é publicado para uma [implantação dependente de estrutura](/dotnet/articles/core/deploying/#framework-dependent-deployments-fdd) e configura o módulo ASP.NET Core para manipular solicitações de site:</span><span class="sxs-lookup"><span data-stu-id="933d3-126">The following `web.config` file is published for a [framework-dependent deployment](/dotnet/articles/core/deploying/#framework-dependent-deployments-fdd) and configures the ASP.NET Core Module to handle site requests:</span></span>

```xml

  </location>
</configuration>
```
