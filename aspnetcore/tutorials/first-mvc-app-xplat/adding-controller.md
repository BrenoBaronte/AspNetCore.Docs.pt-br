# <a name="adding-a-controller-to-an-aspnet-core-mvc-app-with-visual-studio-code"></a>Adicionando um controlador a um aplicativo ASP.NET Core MVC com o Visual Studio Code

Por [Rick Anderson](https://twitter.com/RickAndMSFT)

[!INCLUDE[adding-controller](../../includes/mvc-intro/adding-controller1.md)]

* No **VS Code**, selecione o ícone **EXPLORER** e, em seguida, pressione Control (clique com o botão direito do mouse) **Controladores > Novo Arquivo** e nomeie o novo arquivo HelloWorldController.cs.
