ASP.NET Core Identity Adiciona a funcionalidade de logon da interface do usuário para ASP.NET Core aplicativos Web. Para proteger APIs Web e SPAs, use um dos seguintes:

* [Azure Active Directory](/azure/api-management/api-management-howto-protect-backend-with-aad)
* [Azure Active Directory B2C](/azure/active-directory-b2c/active-directory-b2c-custom-rest-api-netfw) (Azure ad B2C)
* [IdentityServer4](https://identityserver.io)

IdentityServer4 é uma estrutura de OpenID Connect e OAuth 2,0 para ASP.NET Core. IdentityO Server4 habilita os seguintes recursos de segurança:

* AaaS (autenticação como serviço)
* Logon único/logoff (SSO) em vários tipos de aplicativos
* Controle de acesso para APIs
* Gateway de Federação

Para obter mais informações, consulte [Bem-vindo ao Identity Server4](https://docs.identityserver.io/en/latest/index.html).
