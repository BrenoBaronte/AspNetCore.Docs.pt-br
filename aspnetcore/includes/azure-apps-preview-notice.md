> [!IMPORTANT]
> **ASP.NET Core 3,0 com Azure App Service**
>
> O ASP.NET Core 3,0 ainda não foi implantado no serviço Azure App. Esperamos fornecer ASP.NET Core 3 sobre Azure App serviço em breve. Para hospedar um aplicativo que usa um ASP.NET Core 3,0:

* Trate ASP.NET Core 3,0 como uma versão de visualização para Azure App implantação do serviço.
* Confira [Implantar a versão prévia do ASP.NET Core para o Serviço de Aplicativo do Azure](xref:host-and-deploy/azure-apps/index#deploy-aspnet-core-preview-release-to-azure-app-service).
