> [!IMPORTANT]
> **Versões prévias do ASP.NET Core com o Serviço de Aplicativo do Azure**
>
> Versões prévias do ASP.NET Core não são implantadas para o Serviço de Aplicativo do Azure por padrão. Para hospedar um aplicativo que usa uma versão prévia do ASP.NET Core, veja [Implantar versão prévia do ASP.NET Core para o Serviço de Aplicativo do Azure](xref:host-and-deploy/azure-apps/index#deploy-aspnet-core-preview-release-to-azure-app-service).
