Este tutorial ensina a usar o ASP.NET Core MVC e o Entity Framework Core com controladores e exibições. [ Razor Páginas](xref:razor-pages/index) é um modelo de programação alternativo. Para novos desenvolvimentos, recomendamos Razor páginas sobre MVC com controladores e exibições. Consulte a versão de [ Razor páginas](xref:data/ef-rp/intro) deste tutorial. Cada tutorial aborda um material diferente:

Algumas coisas que este tutorial do MVC tem que o Razor tutorial de páginas não:

* Implementar a herança no modelo de dados
* Executar consultas SQL brutas
* Usar o LINQ dinâmico para simplificar o código

Algumas coisas Razor que o tutorial de páginas tem:

* Usar o método Select para carregar dados relacionados
* Práticas recomendadas para o EF.
