## <a name="sqlite"></a>SQLite

O site do [SQLite](https://www.sqlite.org/) afirma:

> O SQLite é um mecanismo de banco de dados SQL independente, de alta confiabilidade, inserido, completo e de domínio público. O SQLite é o mecanismo de banco de dados mais usado no mundo.

Há muitas ferramentas de terceiros que podem ser baixadas para gerenciar e exibir um banco de dados SQLite. A imagem abaixo é do [Navegador do DB para SQLite](https://sqlitebrowser.org/). Se você tiver uma ferramenta favorita do SQLite, deixe um comentário sobre o que mais gosta dela.
