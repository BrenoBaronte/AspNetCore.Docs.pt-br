# <a name="community-oss-authentication-options"></a>Opções de autenticação de comunidade OSS

Esta página contém as opções de autenticação fornecido pela comunidade de código aberto para o ASP.NET Core. Esta página será atualizada periodicamente como novos provedores de se tornar disponíveis.

# <a name="oss-authentication-providers"></a>Provedores de autenticação de sistemas operacionais

A lista a seguir está classificada alfabeticamente.

| Nome | Descrição |
|:--------------|:------------------|
| [AspNet.Security.OpenIdConnect.Server (ASOS)](https://github.com/aspnet-contrib/AspNet.Security.OpenIdConnect.Server) | Baixa-nível/protocolo primeiro OpenID Connect estrutura do server Core do ASP.NET e OWIN/Katana |
| [IdentityServer4](https://identityserver.io/) | Estrutura de OpenID Connect e OAuth 2.0 para ASP.NET Core - certificado oficialmente a base de OpenID e sob controle do Foundation .NET |
| [OpenIddict](https://github.com/openiddict/openiddict-core) | Fácil de usar OpenID Connect server Core do ASP.NET  |
| [PwdLess](https://github.com/pwdless/pwdless) | Autenticação Simple, sem monitoração de estado passwordless para ASP.NET Core  |

Para obter o provedor adicionado aqui [editar esta página](https://github.com/login?return_to=https%3A%2F%2Fgithub.com%2Faspnet%2FDocs%2Fedit%2Fmaster%2Faspnetcore%2Fsecurity%2Fauthentication%2Fcommunity.md).
