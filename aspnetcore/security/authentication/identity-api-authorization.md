<span data-ttu-id="bd2e2-264">Você também pode configurar os clientes e os recursos por meio de código usando uma sobrecarga do `AddApiAuthorization` que executa uma ação para configurar opções.</span><span class="sxs-lookup"><span data-stu-id="bd2e2-264">You can also configure the clients and resources through code using an overload of `AddApiAuthorization` that takes an action to configure options.</span></span>
